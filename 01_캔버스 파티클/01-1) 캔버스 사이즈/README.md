# 1. width, height

- width 랑 height 는 canvas 의 속성값으로 주는 것이 기본
- css 로 style 하면 이미지를 줄이고 늘이는 것과 같은 효과를 자아냄

# 2. dpr

- device pixel ratio
  - 하나의 css 픽셀을 그리는데 사용되는 장치의 픽셀 수
  - 1px 를 그리는데 실제로 몇 px 을 사용하는가
- dpr = 1 : 1px 을 그리는데 하나의 장치만 사용
- dpr = 2 : 1px 을 그리는데 네 개의 장치(2\*2)를 사용
- dpr 이 높을수록 선명할 것
